# Plan: Fix Backend Start Issue (EADDRINUSE)

## Issue
`EADDRINUSE: address already in use :::5000` - Port 5000 is occupied by another process.

## Solution Steps

### 1. Find Process Using Port 5000
Run on Windows:
```powershell
netstat -ano | findstr :5000
```

This returns the PID of the process using port 5000.

### 2. Kill the Process
```powershell
taskkill /PID <PID> /F
```

### 3. Start the Server
```bash
cd server && npm run dev
```

## Implementation Commands
1. Find PID on port 5000 (PID: 26104)
2. Taskkill blocked by permissions - ALTERNATIVE: Change port to 5001

## Final Solution
Edit `server/.env`:
- Change `PORT=5000` to `PORT=5001`
- Also change the PORT in `server/src/index.js` if needed

After change, run: `cd server && npm run dev`
