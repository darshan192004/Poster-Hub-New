# Plan: Transform Poster_Hub to Canva.in-style Design Platform

## Current State
- Existing app with basic templates in 5 categories (Birthday, Business, Sale, Wedding, Festivals)
- 20+ editable templates with drag-and-drop text editing
- Basic editor with fonts, colors, alignment, AI suggestions

## Goal
Transform into Canva.in-style platform with wide template categories and professional editing

## Additional Tasks Based on Selection

### Backend Changes: Template Database

**New Files/Changes:**
| File | Purpose |
|------|---------|
| `server/src/models/Template.js` | Template schema |
| `server/src/routes/templateRoutes.js` | Template CRUD API |
| `server/src/controller/templateController.js` | Template logic |

**Template Schema:**
```javascript
{
  name: String,
  category: String,
  subcategory: String,
  tags: [String],
  style: String,
  dimensions: { width: Number, height: Number },
  backgroundStyle: String,
  isPremium: Boolean,
  zones: [{
    id: String,
    text: String,
    x: Number, y: Number,
    fontSize: Number,
    fontFamily: String,
    color: String
  }],
  createdAt: Date
}
```

**API Endpoints:**
- GET /api/templates - List all templates
- GET /api/templates/:category - By category
- GET /api/templates/search?q= - Search
- POST /api/templates - Admin add (with auth)
