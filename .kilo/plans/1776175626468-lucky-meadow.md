# Plan: Fix EADDRINUSE Error on Port 5000

## Problem
Port 5000 is already in use - another process is listening on it.

## Solution

### Step 1: Identify process using port 5000
Run the following command to find the PID using port 5000:
```bash
netstat -ano | findstr :5000
```

### Step 2: Kill the process
Use the PID found in Step 1 to terminate it:
```bash
taskkill /PID <PID> /F
```

### Step 3: Restart the server
```bash
cd server && npm run dev
```

## Alternative
If Step 1/2 doesn't work, use Task Manager or Windows Settings to find and stop the application using port 5000.
